---
name: wasm
description: WASM agent management -- list running agents, check status, browse gallery
---

Manage WASM agents:

1. Call `mcp__plugin_ruflo-core_ruflo__wasm_agent_list` to show all running WASM agents
2. Call `mcp__plugin_ruflo-core_ruflo__wasm_gallery_list` to show available gallery agents
3. Show agent IDs, status, and sandbox resource usage
