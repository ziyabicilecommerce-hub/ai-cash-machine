## Design Principles (Brutal Edition)

**1. Hero Section (3-second rule)**
- Visitor has 3 seconds to understand your value
- If unclear = bounce
- Hero needs: Headline + subheadline + image + CTA

**2. Color Psychology**
- Red CTA buttons: +35% clicks
- Blue backgrounds: +25% trust feeling
- White space: Increases perception of luxury
- Contrast matters: Eye drawn to colored elements

**3. Typography**
- Large headlines (36px+) = authority
- Easy-to-read body (16px+, line-height 1.6)
- Limit fonts to 2 (one serif, one sans-serif)
- Bold key phrases (makes scannable)

**4. Form Friction**
- Every field = 35% drop in conversions
- Ideal: 1-3 fields maximum
- Email capture first, then upsell details
- Mobile: One-column, not two

**5. Social Proof Zone**
- Place testimonials in "trust zone" (right side, mid-page)
- Use faces + specific results ("$10K earned")
- Video testimonials = 3x text
- Numbers > generic praise

**6. CTA Placement**
- Primary CTA: Above the fold (must be visible immediately)
- Secondary CTAs: Every 300px down page
- Button text: Power words ("Claim", "Access", "Get", not "Submit")
- High contrast: Red on white beats subtle colors

**7. Visual Hierarchy**
- Largest = most important
- Whitespace guides eyes
- Color draws attention
- Movement (video/animation) stops scrolling

**8. Mobile Optimization**
- 60% of traffic is mobile
- Thumb zone: Bottom center for CTAs
- Touch targets: 44×44px minimum
- Load time: < 2 seconds or -25% conversions

**9. Scarcity Communication**
- Countdown timer (high contrast red)
- "Only X spots left" (large, bold)
- Social proof numbers (impressive formatting)
- Urgency graphics (warning yellow/red)

**10. Objection Handling**
- Place objection answers near related content
- "Why you?" (credibility section)
- "What if it doesn't work?" (guarantee/refund)
- "Is this for me?" (qualification section)
- "What's the catch?" (transparency)

## Real Example: $497 E-Book Landing Page

**Brutal Design Strategy:**

1. **HERO** (Above fold)
   - Image: Successful freelancer making $10K/month (Higgsfield-generated)
   - Headline: "The ONE Copywriting Secret..."
   - Subheadline: "Reverse-engineered from 200+ 6-figure freelancers"
   - CTA Button: RED, "Claim Your Copy Now" (45px tall, center-aligned)

2. **SOCIAL PROOF** (Right side, mid-page)
   - 3 testimonials with faces + names
   - Quote: "$8K earned in first month" (specific number)
   - Video testimonial (3 people, 30 seconds total)
   - "1,247 copies sold" (impressive number)

3. **VALUE PROP** (3 bullets, bold)
   - "The exact 3-formula system to land $5K+ projects"
   - "Complete templates you can use today"
   - "Lifetime access to private community"

4. **OBJECTION HANDLERS** (Mid-page)
   - "Is this just another e-book?" → NO - includes personal coaching
   - "I'm not a copywriter" → Doesn't matter - 0 experience needed
   - "What if it doesn't work?" → 100% refund guarantee

5. **FINAL CTA** (Bottom of page)
   - Urgency: "Only 47 spots at $497 | Price goes up tomorrow"
   - Button: "Get Instant Access Now"
   - Repetition of main benefit

6. **MOBILE** (Optimized for 60% of traffic)
   - Vertical stacking (not side-by-side)
   - Larger text (18px+ body, 32px+ headlines)
   - One-column form (email + name only)
   - Large button (44px height, full width)
   - Fast load (image optimized, < 2 seconds)

## Design Specifications

- **Layout:** Single column (mobile-first responsive)
- **Width:** Max 600px for text (optimal reading line length)
- **Colors:** Red (#FF0000) CTA, white background, dark gray text (#222)
- **Typography:** 32px headlines, 16px body, 18px subheadlines
- **Spacing:** 32px/16px rhythm system
- **Forms:** 1-3 fields max (email, name, optional)
- **Images:** Higgsfield-generated (hero + testimonials)
- **Load time:** < 2 seconds (optimize images to 50KB max)
- **Mobile:** Full width, thumb-zone CTAs
- **Accessibility:** 4.5:1 contrast ratio, alt text on all images

## Tech Stack Recommendations

- **HTML/CSS:** Simple, no framework (lightest load time)
- **Higgsfield:** For hero image + testimonials
- **Form service:** Webflow, Leadpages, or custom serverless
- **Analytics:** GA4 (track every click, scroll depth)
- **A/B testing:** Unbounce, Instapage, or Webflow A/B
- **Email:** Convertkit, ActiveCampaign (with automation)

## Workflow

1. Define value prop (8 words max)
2. Generate hero image (Higgsfield)
3. Write copy (headline, subheadline, 3 bullets)
4. Design layout (wireframe → Figma → code)
5. Generate videos (testimonials via Higgsfield)
6. Set up forms (email capture → email sequence)
7. A/B test variants (3-5 different hero designs)
8. Monitor analytics (conversion rate, bounce rate, time on page)
9. Optimize: (repeat highest-lift changes)

## Red Flags (What NOT to Do)

- Long copy (mobile users won't read it)
- Multiple CTAs that are equal size (confusion)
- No social proof (skepticism)
- Unclear value prop (what's your main benefit?)
- No mobile optimization (60% bounce rate)
- Weak CTA copy ("Submit" = lame, "Claim now" = strong)
- Too many colors (unfocused eyes)
- Slow load time (each second = 7% conversion drop)
- No urgency (visitors procrastinate forever)
- No objection handling (unanswered objections = bounces)
