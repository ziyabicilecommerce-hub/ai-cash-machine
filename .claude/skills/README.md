# Claude Code Skills & Slash-Commands

Dieses Repo enthält **348 echte Claude-Code-Skills** (`.claude/skills/<name>/SKILL.md`)
und zu jedem einen passenden **Slash-Command** (`.claude/commands/<name>.md`).

## Zwei Wege, sie zu nutzen

1. **Automatisch (Skills):** Claude aktiviert einen Skill selbst, wenn deine
   Anfrage zur Skill-Beschreibung passt.
2. **Explizit (Slash-Commands):** Tippe `/<name> <deine anfrage>`, z. B.
   `/copywriting schreib mir eine Hook für mein KlickTipp-Reel`. Der Command
   wendet gezielt den jeweiligen Skill an.

## Herkunft & Lizenz

Die Skills stammen aus drei quelloffenen, MIT-lizenzierten Repositories:

- [alirezarezvani/claude-skills](https://github.com/alirezarezvani/claude-skills) (346 Skills — Marketing, Engineering, C-Level, Compliance …)
- [ruvnet/ruflo](https://github.com/ruvnet/ruflo) (`ruflo` — Multi-Agent-Orchestrierung, `npx ruflo`)
- [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) (`ui-ux-pro-max` — Design-Intelligence-Datenbank)

Symlink-Spiegelungen und Aliase wurden aufgelöst und dedupliziert – jeder
Skill liegt genau einmal als echte Datei vor.
