# Claude Code Skills & Slash-Commands

Dieses Repo enthält **346 echte Claude-Code-Skills** (`.claude/skills/<name>/SKILL.md`)
und zu jedem einen passenden **Slash-Command** (`.claude/commands/<name>.md`).

## Zwei Wege, sie zu nutzen

1. **Automatisch (Skills):** Claude aktiviert einen Skill selbst, wenn deine
   Anfrage zur Skill-Beschreibung passt.
2. **Explizit (Slash-Commands):** Tippe `/<name> <deine anfrage>`, z. B.
   `/copywriting schreib mir eine Hook für mein KlickTipp-Reel`. Der Command
   wendet gezielt den jeweiligen Skill an.
